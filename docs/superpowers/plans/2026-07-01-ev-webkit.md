# EV::WebKit Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** An asynchronous, in-process browser-automation library on WebKitGTK 6.0 (GTK4), driven from Perl via GObject Introspection, with an EV-callback API (Firefox::Marionette-inspired, WebKit-native).

**Architecture:** Pure Perl over `Glib::Object::Introspection` (no XS). Native WebKit calls for navigation/snapshot/PDF/cookies/proxy/settings; DOM find+interact through a single injected-JavaScript bridge built on `call_async_javascript_function` (WebKitGTK 6.0 removed the DOM API). A private `Xvfb` gives headless rendering. Runs on `EV` + `EV::Glib`; every async op takes a `sub { my ($result, $err) = @_ }` callback.

**Tech Stack:** Perl 5.10+, EV, EV::Glib, Glib, Glib::Object::Introspection, Cpanel::JSON::XS; runtime typelibs WebKit-6.0 / Gtk-4.0 / Gdk-4.0 / JavaScriptCore-6.0 / Soup-3.0; Xvfb.

## Global Constraints

> **ARCHITECTURE REVISION (2026-07-01): bring-your-own-display.** The module no
> longer spawns or kills Xvfb. `EV::WebKit::Display` and **Task 2 are REMOVED**.
> `new()` uses the ambient `$ENV{DISPLAY}` (or `display => ':N'`) and dies with
> guidance if none; `quit()`/`DESTROY` destroy only the GTK window and never touch
> an X server. **All browser tests run under `xvfb-run -a`** (one shared Xvfb per
> suite run): read every `perl -Ilib t/XX.t` command below as
> `xvfb-run -a perl -Ilib t/XX.t`, and CI runs `xvfb-run -a make test`. Ignore any
> `headless`/`visible`/auto-Xvfb wording in later tasks. Rationale: GTK4 is
> process-global single-display; a per-instance Xvfb killed on quit() trips Xlib's
> "connection broken -> exit(1)" handler and blocks multi-instance. Verified: full
> suite passes and exits 0 under `xvfb-run -a`; multiple instances per process work.

- Perl floor: `use v5.10;` (match the EV::* family, e.g. EV::Pg).
- No XS, no pkg-config/`-dev` deps. Runtime deps are typelibs + `Xvfb`, checked at load with a clear message; tests `skip_all` when absent.
- Callback convention everywhere: `$cb->($result, $err)`. On success `$err` is undef; on failure `$result` is undef and `$err` is a cleaned string.
- Author attribution: `vividsnow` only. **No AI/Co-Authored-By lines in commits.**
- POD: plain ASCII (no em-dash, no unicode).
- Platform: **Linux** for v1.
- Namespace: `EV::WebKit`, `EV::WebKit::Element`, `EV::WebKit::Display`.
- JSON codec: prefer `Cpanel::JSON::XS`, fall back to core `JSON::PP`.

package EV::WebKit::Display;
use v5.10; use strict; use warnings;
use POSIX ':sys_wait_h';

sub new {
    my ($class, %o) = @_;
    my ($w, $h) = ($o{width} || 1280, $o{height} || 1024);
    local $^F = 10;                       # keep pipe fds OFF close-on-exec so Xvfb inherits the write end
    pipe(my $rd, my $wr) or die "pipe: $!";
    my $pid = fork // die "fork: $!";
    if (!$pid) {                          # child: exec Xvfb, report chosen display on fd
        close $rd;
        my $fd = fileno $wr;
        exec('Xvfb', '-displayfd', $fd, '-screen', '0', "${w}x${h}x24", '-nolisten', 'tcp')
            or POSIX::_exit(127);
    }
    close $wr;
    my $num = '';
    # Xvfb prints the display number to the fd once ready
    local $SIG{ALRM} = sub { die "Xvfb startup timeout\n" };
    alarm 15;
    while (my $line = <$rd>) { $num .= $line; last if $num =~ /\n/ }
    alarm 0;
    close $rd;
    chomp $num;
    die "Xvfb failed to report a display\n" unless $num =~ /^\d+$/;
    my $self = bless { pid => $pid, display => ":$num", owner => $$ }, $class;
    $ENV{DISPLAY}     = $self->{display};
    $ENV{GDK_BACKEND} = 'x11';
    return $self;
}

sub display { $_[0]{display} }

sub stop {
    my $self = shift;
    my $pid = delete $self->{pid} or return;
    kill 'TERM', $pid;
    for (1..50) { return if waitpid($pid, WNOHANG) == $pid; select undef,undef,undef,0.02 }
    kill 'KILL', $pid; waitpid($pid, 0);
}

sub DESTROY { my $self = shift; $self->stop if ($self->{owner} // 0) == $$ }

1;
```

use EV;
use EV::Glib;                # integrates the GLib main loop into EV
use EV::WebKit::Display;

sub new {
    my ($class, %o) = @_;
    _setup() or die "EV::WebKit: required typelibs unavailable\n";
    my $self = bless {
        timeout   => $o{timeout} // 30,
        on_error  => $o{on_error},
        on_load   => $o{on_load},
        pending   => undef,       # pending nav [cb, timer]
    }, $class;

    my ($w, $h) = @{ $o{window} || [1280, 1024] };
    unless ($o{visible} || $o{display}) {
        $self->{display} = EV::WebKit::Display->new(width=>$w, height=>$h);
    } elsif ($o{display}) {
        $ENV{DISPLAY} = $o{display}; $ENV{GDK_BACKEND} //= 'x11';
    }
    _init_gtk();   # display now exists -> safe to init GTK

    my $session = $self->{session} =
        (defined $o{ephemeral} ? $o{ephemeral} : 1)
        ? WebKit::NetworkSession->new_ephemeral
        : WebKit::NetworkSession->new(undef, undef);
    my $ucm = $self->{ucm} = WebKit::UserContentManager->new;
    my $view = $self->{view} = Glib::Object::new('WebKit::WebView',
        'network-session' => $session, 'user-content-manager' => $ucm);

    if (defined $o{user_agent}) { $view->get_settings->set('user-agent', $o{user_agent}) }

    my $win = $self->{win} = Gtk4::Window->new;
    $win->set_default_size($w, $h);
    $win->set_child($view);
    $win->present;

    $view->signal_connect('load-changed' => sub {
        my (undef, $ev) = @_;
        $self->_finish_nav(undef) if $ev eq 'finished';
    });
    $view->signal_connect('load-failed' => sub {
        my (undef, undef, $uri, $gerr) = @_;
        $self->_finish_nav("load failed: $uri: ".(ref $gerr ? $gerr->message : $gerr));
        return 1; # handled
    });
    return $self;
}

sub _finish_nav {
    my ($self, $err) = @_;
    my $p = delete $self->{pending} or return;
    $p->[1]->stop if $p->[1];
    my $cb = $p->[0];
    if ($err) { $cb->(undef, $err) if $cb }
    else      { $cb->(1, undef) if $cb; $self->{on_load}->() if $self->{on_load} }
}

sub _start_nav {
    my ($self, $cb) = @_;
    # replace any in-flight nav with a timeout error
    $self->_finish_nav('superseded') if $self->{pending};
    my $timer = EV::timer($self->{timeout}, 0, sub { $self->_finish_nav('timeout') });
    $self->{pending} = [$cb, $timer];
}

sub go {
    my ($self, $uri, $cb) = @_;
    $self->_start_nav($cb);
    $self->{view}->load_uri($uri);
    return $self;
}

sub load_html {
    my ($self, $html, $cb) = @_;
    $self->_start_nav($cb);
    $self->{view}->load_html($html, undef);
    return $self;
}

sub uri        { $_[0]{view}->get_uri }
sub title      { $_[0]{view}->get_title }
sub is_loading { $_[0]{view}->is_loading ? 1 : 0 }

sub quit {
    my $self = shift;
    $self->_finish_nav('quit') if $self->{pending};
    delete $self->{view}; delete $self->{win};
    delete $self->{ucm};  delete $self->{session};
    $self->{display}->stop if $self->{display};
    return;
}
sub DESTROY { my $self = shift; eval { $self->quit } }
```

sub wait_for {
    my ($self, $sel, %o) = (shift, shift);
    my $cb = pop;
    %o = @_;
    my $deadline = $o{timeout} // $self->{timeout};
    my $interval = $o{interval} // 0.05;
    my $visible  = $o{visible};
    my $elapsed  = 0;
    my $w; # keep watcher alive
    my $tick; $tick = sub {
        $self->find($sel, sub {
            my ($el, $err) = @_;
            return $cb->(undef, $err) if $err;
            my $accept = sub {                      # resolve now
                undef $w; $cb->($el, undef);
            };
            if ($el && $visible) {
                return $el->is_visible(sub {
                    return $accept->() if $_[0];
                    _reschedule($self, \$w, \$elapsed, $interval, $deadline, $tick, $cb);
                });
            }
            return $accept->() if $el;
            _reschedule($self, \$w, \$elapsed, $interval, $deadline, $tick, $cb);
        });
    };
    $tick->();
    return $self;
}

sub _reschedule {
    my ($self, $wref, $eref, $interval, $deadline, $tick, $cb) = @_;
    if ($$eref >= $deadline) { $$wref = undef; return $cb->(undef, 'timeout') }
    $$eref += $interval;
    $$wref = EV::timer($interval, 0, sub { $tick->() });
}

=head1 NAME

EV::WebKit - async WebKitGTK 6.0 (GTK4) browser automation on EV

=head1 SYNOPSIS

    use EV;
    use EV::WebKit;

    my $b = EV::WebKit->new(
        headless   => 1,                       # default: private Xvfb
        user_agent => 'Mozilla/5.0 ...',
        on_console => sub { warn "console: $_[0]" },
        on_error   => sub { warn $_[0] },
    );

    $b->go('https://example.com', sub {
        my (undef, $err) = @_;
        die $err if $err;
        $b->find('h1', sub {
            my ($el) = @_;
            $el->text(sub { print "H1: $_[0]\n" });
        });
        $b->screenshot('shot.png', sub { EV::break });
    });

    EV::run;

=head1 DESCRIPTION

EV::WebKit drives an in-process WebKitGTK 6.0 web view for headless browser
automation, exposing a WebKit-native, asynchronous, EV-callback API inspired by
Firefox::Marionette. It requires the WebKit-6.0 / Gtk-4.0 / JavaScriptCore-6.0 /
Soup-3.0 GObject-Introspection typelibs and Xvfb; there is no XS component.

DOM access is performed by injecting JavaScript through WebKit's
call_async_javascript_function; found elements are returned as
L<EV::WebKit::Element> handles.

=head1 METHODS

... (document new, go, load_html, uri, title, is_loading, script,
script_async, find, find_all, wait_for, screenshot, pdf, set_user_agent,
user_agent, settings, set_cookie, cookies, clear_cookies, set_proxy,
mock_scheme, quit -- each with its callback signature) ...

=head1 EVENTS

Constructor callbacks: on_load, on_error, on_console, on_dialog, on_policy.

=head1 REQUIREMENTS

WebKitGTK 6.0, GTK4, JavaScriptCore 6.0, libsoup3 (GObject-Introspection
typelibs) and Xvfb. Linux only.

vividsnow

the same terms as Perl itself.
